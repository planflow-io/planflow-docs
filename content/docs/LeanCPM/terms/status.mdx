---
title: Status
---
The options for Task Status are:

| Status         | Scope | Condition                          |
| -------------- | ----- | ---------------------------------- |
| Not Ready      | Tasks | blocked by dependencies            |
| Ready to Start | Tasks | No longer blocked by dependencies  |
| Started        | Task  | User set                           |
| Finished       | Task  | User Set                           |
| Completed      | Task  | User Set to Finish at least 2w ago |
