---
title: Expected Dates
description: Anticipated start and finish for
---
The flexibility of LeanCPM means the user can show tasks as starting and finishing on any date they choose. 

<Callout title="Calculating Schedules" type="info">
Only the expected dates of the first and last activities in a schedule are considered when Calculating early and late dates.
</Callout>

## Expected Start
The date the user expects the task to start

## Expected Finish 

<Callout title="Legacy CPM" type="warn">
In legacy CPM, there is an "Expected Finish" field to help calculate the Remaining Duration. Expected Finish is only valid for "In Progress" activities. There is no concept of "Expected Start" in legacy CPM.
</Callout>
